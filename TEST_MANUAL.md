# 🧪 Teste Manual - Debug

## Por favor, faça este teste:

1. Abra: http://localhost:3000
2. Abra o Console do navegador (F12)
3. Procure por estas mensagens:

```
🔍 Search normalizer: Módulo carregado
✅ Search normalizer: Input de busca encontrado
```

4. Clique no campo de busca (ícone da lupa)
5. Digite: "usuário"
6. Me diga:
   - ✅ O texto mudou para "usuario"? 
   - ❌ O texto ficou como "usuário"?
   - ❓ Qual mensagem apareceu no console?

## Se não aparecer nenhuma mensagem:

O módulo não está sendo carregado. Vou precisar usar outra abordagem.

## Se aparecer "Input não encontrado":

Preciso ajustar os seletores CSS para encontrar o input correto.

## Me envie:

1. Screenshot ou texto das mensagens do console
2. O que acontece quando você digita "usuário"
