<h3 class="wp-block-heading"><strong>Passo 1: Como Instalar o Sistema Facity</strong></h3>



<ul class="wp-block-list">
<li><strong>1.1. Baixe o Sistema</strong>: A primeira coisa a fazer é baixar o sistema Facity no seu computador ou dispositivo. Dependendo do seu modelo de negócio, o Facity oferece versões específicas que podem ser baixadas diretamente no site da Facity.</li>



<li><strong>1.2. Realize a Instalação</strong>: Após o download, siga as instruções para instalar o sistema. O processo é simples e intuitivo, mas se surgir alguma dúvida, há tutoriais disponíveis para ajudar.</li>



<li><strong>1.3. Configurações Iniciais</strong>: Quando a instalação estiver concluída, o sistema vai pedir algumas configurações iniciais. Isso inclui a inserção de dados do seu negócio, como nome, endereço e tipo de operação. Preencha essas informações com cuidado, pois elas são importantes para o funcionamento correto do sistema.</li>
</ul>

<h3 class="wp-block-heading"><strong>Passo 6: Como Cadastrar seu Estoque no Facity Controle</strong></h3>



<p></p>



<ol class="wp-block-list">
<li><strong>Acesse o sistema</strong>: Abra o Facity Controle e faça login.​</li>



<li><strong>Vá para &#8220;Estoque&#8221; &gt; &#8220;Matérias-Primas&#8221;</strong>: No menu, selecione &#8220;Estoque&#8221; e depois &#8220;Matérias-Primas&#8221;.​</li>



<li><strong>Adicione uma nova matéria-prima</strong>: Clique em &#8220;Novo&#8221;, preencha o nome, quantidade e unidade, depois salve.​</li>



<li><strong>Vincule a matéria-prima a um produto</strong>:
<ul class="wp-block-list">
<li>Acesse &#8220;Produtos&#8221; e edite o produto desejado.​</li>



<li>Vá para &#8220;Ficha Técnica&#8221; e clique em &#8220;Novo&#8221;.​</li>



<li>Selecione a matéria-prima, defina a quantidade a ser descontada do estoque e salve.​</li>
</ul>
</li>



<li><strong>Registre entradas no estoque</strong>: Em &#8220;Estoque&#8221;, vá para &#8220;Entradas&#8221;, clique em &#8220;Novo&#8221;, selecione &#8220;Matérias-Primas&#8221; e insira as informações da nova entrada.​</li>
</ol>



<p>Seguindo esses passos, você poderá gerenciar seu estoque no Facity Controle de forma eficiente.</p>

<p class="has-medium-font-size"><strong> </strong> <strong>Passo 12: Como gerar um relatório de vendas no Facity Controle</strong></p>



<p></p>



<ol class="wp-block-list">
<li><strong>Acesse o sistema</strong>: Faça login no Facity Controle.​</li>



<li><strong>Vá para &#8220;Relatórios&#8221; &gt; &#8220;Vendas&#8221;</strong>: No menu lateral, clique em &#8220;Relatórios&#8221; e depois em &#8220;Vendas&#8221;.​</li>
