<p><strong>Transferir uma Mesa</strong>:<br>Quando precisar transferir uma mesa, basta pressionar <strong>ALT + T</strong>.</p>



<p><strong>Bloquear uma Mesa</strong>:<br>Para bloquear uma mesa , pressione <strong>ALT + B</strong>.</p>
